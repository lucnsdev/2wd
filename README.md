Um Projeto para controlar um robô via bluetooth.

Desenho virtual da placa de circuito feito no KiCad e desenhado fisicamente usando o LaserRGBL com uma CNC a Laser.
Mas é possivel construir usando componentes pré prontos. 
Um exemplo simples está na imagem do desenho do circuito usando placas tradicionais e modulos para Arduino.
