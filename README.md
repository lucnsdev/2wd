Um Projeto para controlar um robô via bluetooth.

Desenho feito no KiCad e desenhado usando o LaserRGBL.
Foi usado uma CNC a Laser para desenhar fisicamente a placa de circuito.
Mas é possivel construir usando componentes pré prontos. Um exemplo simples está na imagem do desenho do circuito usando placas tradicionais.
